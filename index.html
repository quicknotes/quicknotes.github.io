Quick Notes

A fast, distraction-aware note-taking extension for Chrome

Install from Chrome Web Store

What it is

A floating notepad that lives above any webpage. Press Alt+N on any page and start typing — auto-saves as you go. Supports Markdown, images, tables, code blocks, color tags, split view, searchable history, and optional Google Drive sync. 100% local by default. No accounts. No telemetry. No ads.

Features

⚡ Floating panel

Drag anywhere, resize freely, persists across tabs.

📝 Markdown preview

Headings, tables, code, links, checklists — all roundtrip.

🖼 Images & GIFs

Paste or drag-drop. GIFs animate in preview.

↔ Split view

Horizontal or vertical, two notes side by side.

🔍 Search

Ctrl+F across all notes with snippets.

🎨 Themes

Dark & light + custom palette overrides.

☁ Drive sync

Opt-in, uses hidden app-data folder. Private.

🔒 Privacy first

All data local by default. Zero telemetry.
